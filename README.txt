Movie Trailer Website

This is a server-side code to store a list of movies, including box art imagery and a movie trailer URL. The code generates a static web page allowing visitors to browse movies and watch the trailers.
 
Requirements

This code requires Python installed.


Installation

•	Download movietrailerwebsite.zip
•	Decompress movietrailerwebsite.zip

Usage

•	$ cd movietrailerwebsite
•	$ python entertainment_center.py
